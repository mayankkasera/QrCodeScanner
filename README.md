# QrCodeScanner

This is simple QR code scanner with Room Database. 100% written in Kotlin.
App Features: 
* Scan any QR code or Bar Code.
* Automatically saved in Local Database. 
* Flash Light On/Off Feature
* Set any QR Result as Favourite.
